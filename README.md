# 🚁 Disaster Drone Rescue AI

An interactive simulation where AI-controlled drones search disaster zones to locate survivors using reinforcement learning concepts.

This project demonstrates how autonomous agents can explore unknown environments and coordinate search operations efficiently.

---

# 🌍 Problem Statement

In real disaster situations such as earthquakes, floods, or collapsed buildings, rescue teams must quickly locate survivors.

Autonomous drones can explore dangerous areas faster and safer than humans.

This project simulates a **drone rescue system** where drones navigate a grid-based environment to detect survivors.

---

# 🧠 Key Features

- Multi-Drone Coordination
- Grid-Based Disaster Simulation
- Survivor Detection System
- Reinforcement Learning Environment
- Interactive Web Dashboard

# 🗺️ Environment Design

The simulation environment consists of:

- Drone agents
- Disaster grid map
- Survivors hidden in the environment
- Obstacles and danger zones

Each drone can:

- Move in four directions
- Scan nearby cells
- Explore unknown areas

---

# 🎮 Actions

Drone agents can perform the following actions:

| Action | Description |
|------|-------------|
| UP | Move upward |
| DOWN | Move downward |
| LEFT | Move left |
| RIGHT | Move right |
| SCAN | Detect nearby survivors |

---

# 🏆 Objective

The main goal is to:

- Maximize rescue efficiency
